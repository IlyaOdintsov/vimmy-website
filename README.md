# Portfolio Website — Frontend Developer

[vimmy.ru](https://vimmy.ru)

Персональный сайт‑портфолио для демонстрации проектов и навыков во frontend‑разработке.

## Технологии

- React, React Router Dom, Vite

- TypeScript, ESLint, Prettier

- Tailwind CSS, Lucide React, Framer Motion

- Zustand

- i18next, Next‑themes, @emailjs/browser

## Основные возможности

- Адаптивный дизайн для мобильных, планшетов и десктопов  
- Анимированные переходы и интерактивные элементы (Framer Motion + CSS‑анимации)  
- Поддержка светлой и тёмной тем (Next‑themes)  
- Локализация интерфейса на несколько языков (i18next, react‑i18next)  
- Удобная навигация по разделам (React Router Dom)  
- Отправка форм связи через EmailJS без бэкенда  

## Установка и запуск

1. Клонировать репозиторий:  
   ```bash
   git clone https://github.com/IlyaOdintsov/vimmy-website.git
   ```

2. Перейти в папку проекта:  
   ```bash
   cd frontend
   ```

3. Установить зависимости:  
   ```bash
   yarn install
   ```

4. Запустить режим разработки:  
   ```bash
   yarn dev
   ```

5. Открыть в браузере `http://localhost:5173`.
